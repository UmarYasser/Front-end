Media Page

HTML has several media features, like images, audio and videos

We add images using the tag <img src=""> and the image source from the directory

We can add width and height property after "src=" to resize the image as we wish

Or add "alt", to display a message of the image if it doesn't load, it indicats what the image was about

and "title", to display text when the image is hovered on, try it on that picture.

Lastly <figure> and <figcaption> to describe an image in a better way than using a <p> element around it

Audio files are added using the tag <audio ...> and add some controls and/or alternative sources

Audio Contorls include: "autoplay" for autoplaying the file one the page starts

"muted", always used for autoplay for better user experince

"loop" for repeating the file automatically

- We add alternative sources for a file to ensure that the browser supports at least one of them.
that means that if the browser doesn't support the 1st source it checks the 2nd and 3rd source to play the file.

Syntax:

     <audio controls loop muted>
        <source src="music.mp3" type="audio/mpeg">
        <source src="music.wav" type="audio/wav">
        <source src="music.ogg" type="audio/ogg">
    </audio>

Example

Video files are added similarly to audio files with the concept of alternative sources but has more options...

Like "poster" which displays an image on the thumbnail of the video before it's played, which is set to an image source

Or "preload" which tells the browser how to load the file, which can be set to
'none': don't load it at all
'lazy': load it as the user plays it, which reduces the load time of the whole page
'auto': load it all even before the user plays it

And "track" which are the subtitles. Takes 'kind= "subtitles"' 'src="example.vtt"' and label that corresponds to the language or the option that will be displayed to the user

Syntax:
        <video controls muted width="100%" poster="img.jpg" preload="none">
            <source src="video.mp4" type="video/mp4">
            <source src="video.webm" type="video/webm">
            <source src="video.ogg" type="video/ogg">
    
            <track default src="en.vtt" kind="subtitles" label="English">
            <track src="ar.vtt" kind="subtitles" label="Arabic">
       </video>
