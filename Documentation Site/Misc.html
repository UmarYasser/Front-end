Miscellaneous Page

HTML has some elements that doesn't go under any category since they're in the miscellaneous page.

These Elements include:

Horizontal Rule:
Written as <hr> and makes a thick line to seprate parts of the page, like the one at the end of this page.

Line Break:
Written as <br> and acts like a new line to add between elements or in p elements which remove any whitespace written in them.

Embed:
They add a whole website or a resource inside our page like this

Object:
Acheive the same thing as embed but is more versitle as it can add pdfs or flash files
but has compatiablity issues with old browsers

Iframe:
Achevive similar goals to object & embed but is preferred as it doesn't use java applets or plug-ins which aren't supported by modern browsers or mobile
