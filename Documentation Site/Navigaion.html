Navigation Page

HTML Features links that are essential to any website.

The Most Important one is <a> which stands for anchor link that can link to a page in the same website like the link to home page in any page of this documention or the other websites.

Anchor link require a hyper reference or 'href' to the location they're referencing like that
<a href=" ">.

HTML also features <nav> to specify to the developers, search engines and screen readers that this's the area for navigation around the website, while not affecting the appearance of the links. Example of it is the group of links in the home page.

HTML also features linking contact information like email or telephone numbers
which can be done by adding 'mailto:' or 'tel:' at the beginning of href.
