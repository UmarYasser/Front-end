Lists Page

HTML features 3 types to lists: ordered, unordered and defintion. They can be understood by their name.

Ordered lists are written using <ol> enclosing the whole list and <li> enclosing an element, Like:

  1. C
  2. C#
  3. C++
  4. Python 
Syntax:
        <ol>
            <li>
                C
            </li>
            <li>
                C#
            </li>
            <li>
                C++
            </li>
            <li>
                Python
            </li>
        </ol>

*It can have a property "type= "I" " for increasing or "D" for decreasing order and "start" to begin from a certain number

Unordered lists has the same syntax, just using <ul> instead of <ol>

  • C
  • C#
  • C++
  • Python

Defintion Lists give a message under and infront a little bit an upper element like the following:

Faculty of Science
The House of the بصمجية
Syntax:
    <dl>
        <dt>Faculty of Science</dt>
        <dd>The House of the بصمجية</dd>
    </dl>
