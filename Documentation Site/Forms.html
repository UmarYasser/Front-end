Forms Page

Forms are essential in any website using html, to get user info.

Forms are composed of inputs and submit button to send the info to a file or a back-end endpoint.

Types of inputs include: text, number, email, password and telephone number and more.

Inputs take:
Id: if I want to attach something to it like <label>
name:The name of the value sent like: name:Umar
Type which has text, email, password, color, file, search, url or hidden fields that are filled by the website.
And <placeholder> to display what the field is about.

Inputs can be in a form of textarea where input is a large text, or select & option where input is chosen from a list of options

---------------

---------------

Forms takes method and action attribute;
-Method to specify the http request like GET or POST request which differs in their behavior like:
GET exposes the info in the URL while POST doesn't.
Action tells the form where to send the info at, either be a file or a back-end API

Froms can take before the input tags <fieldset> which puts all the input in a frame and <legend> to display text on the top of the frame

Syntax:

<form method="post">
        <fieldset>
            <legend>Form</legend>
        <div>
            <label for="text">user name</label>
            <input id="text" type="text" value="text" required name="user name"> 
        </div>
        <br>
        <div>
            <label for="password">password</label>
            <input id="password" type="password" placeholder="Click here to type your Password" required name="password">
        </div>
        <br>
        <div>
        <input type="submit" value="Send">
        </fieldset>
</form>
