Metadata Page

Any HTML page has two main components: <head> & <body>

The Head can have some options like:

Charset:
Takes an encoding like 'UTF-8' for representing the characters in the screen and support for other languages

Author:
Tells the search engine to promote the site when the given author is searched for

Description:
This is written below the link in any search to give a brief overivew about the site

Viewport:
How the website scales across different devices like mobile phones or laptops

Favicon:
Takes a image source which will be the icon that is display in the tab or a favorite bookmark

Base Tag:
Adds a default link that all link and hrefs gets appended to it
And <target> to determine if a link be opened in a new tab or the same tab
